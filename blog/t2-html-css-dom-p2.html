What is grid based design?

This is a design system where every element is positioned with regard to a grid. It was initially developed in the printing industry, but is now also applied to web page design. If the size of the grid changes the elements should also change by keeping their position relative to the grid.

Whats all the hype about responsive webpages?

These days web pages are viewed on many different devices: desktop PCs, laptops, tablets, and smart phones. These have different screen sizes. Responsive web design is the idea that you develop pages that can automatically resize themselves to fit different screen sizes while maintaining basic usabilty. Twitter Bootstrap is a currently popular framework to achieve this.

Semantic structure

Tags such as header, main, footer, h1, h2, article, aside, section, and nav, convey information about what is the role of tagged data in the web page. They assist with tasks like merging documents and with assitive technologies like screen readers.
