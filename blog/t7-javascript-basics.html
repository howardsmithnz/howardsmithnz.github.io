How does Javascript compare to HTML and CSS?

These technologies do quite different things. HTML provides a web page with structure, CSS gives it style, and Javascript gives it behaviour. HTML seems to be pretty straight forward to learn, CSS is tricky as there are many attributes which change how web components may display, while Javascript is the most difficult to learn well as there is much more depth to this subject.

Explain the difference between accessing data from arrays and object literals

Data stored in an array is stored with respect to order. Data in an object is not stored in any particular order. Therefore we can access array data by using the index of the position e.g.myArray[3]. Data from an object is accessed by using the data's key to retrieve the data value. There are two different notations for this e.g. myObj.keyword or myObj["keyword"]

Explain what functions are and why they are useful

Functions are just units of computer code. By putting frequently used code into a function we can avoid the necessity of typing it out every time - we merely call the function. Also functions help to abstract particlular tasks or ideas therefore simplifying our code.

With an example, explain the differences between 'functional' and 'object oriented' programming

In Object Oriented programming object encapsulate both data and methods(functions of the object). In Functional programming functions are kept separate from data. For example, if we had a game where we wanted to make a ball bounce, using objected oriented programming, we would call a bounce method on a ball object to change the ball's position, like so: Ball.bounce(). In functional programming we would give data representing a balls position to a bounce function, which would then return us the new position of the ball, like so: newBallPos = bounce(oldBallPos).
